# XGEN

XGEN 플랫폼 관련 기술 문서 및 개발 기록.

## 구성

- **xgen-frontend** — Next.js 프론트엔드
- **xgen-backend-gateway** — Rust/Axum API 게이트웨이
- **xgen-core** — Python 코어 서비스
- **xgen-workflow** — 워크플로우 엔진
- **xgen-documents** — 문서 처리 (embedding, retrieval)
- **xgen-mcp-station** — MCP 서버 스테이션
